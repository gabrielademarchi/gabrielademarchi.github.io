Context¶

Elon Musk, billionaire and onwer of Tesla, the electric car company, is on trial after investors claimed a 2018 tweet cost them millions of dollars when a deal did not go ahead.
In the August 7th tweet, Musk wrote that he had "funding secured" to take Tesla private at $420 (341) per share, and that "investor support is confirmed". The tweet sent shares in Tesla soaring, but weeks later they fell back when Musk said the plan was no longer going ahead, causing a significant backlash for the billionaire.
At the jury trial in a San Francisco court on January, he stated that "Just because I tweet something does not mean people believe it or will act accordingly".
Is this really true or is the tycoon underestimating his power to move the financial market?
Lucky for us, we can do an impact analysis to find that answer.

Assumptions¶

  • Parallel Trends Assumption: The treatment and Control Groups are assumed to have the same evolution for the KPI
  • Confounding Policy Change: There must be only one policy or initiative that differentiates the treatment from control groups.

How to Strengthen the assumptions:

  • Use more control groups. The more comparable groups you havem the more you reinforce the causality.
  • Extend the pre period. The more data you have, the more you have to see whether the groups are comparable and would develop similarly or not as a result.
  • Keep post-period to the bare minimum. This minimizes the possibility of othr events affecting the results.

def dickey_fuller(df, col):
    test = adfuller(x = df[col])[1]
    if test < 0.05:
        print("the time series is stationary")
    else:
        print("the time series is not stationary")

Analysis report {CausalImpact}


During the post-intervention period, the response variable had
an average value of approx. 24.3. By contrast, in the absence of an
intervention, we would have expected an average response of 21.37.
The 95% interval of this counterfactual prediction is [19.96, 22.62].
Subtracting this prediction from the observed response yields
an estimate of the causal effect the intervention had on the
response variable. This effect is 2.93 with a 95% interval of
[1.67, 4.34]. For a discussion of the significance of this effect,
see below.


Summing up the individual data points during the post-intervention
period (which can only sometimes be meaningfully interpreted), the
response variable had an overall value of 97.19.
By contrast, had the intervention not taken place, we would have expected
a sum of 85.47. The 95% interval of this prediction is [79.85, 90.5].


The above results are given in terms of absolute numbers. In relative
terms, the response variable showed an increase of +13.71%. The 95%
interval of this percentage is [7.83%, 20.29%].


This means that the positive effect observed during the intervention
period is statistically significant and unlikely to be due to random
fluctuations. It should be noted, however, that the question of whether
this increase also bears substantive significance can only be answered
by comparing the absolute effect (2.93) to the original goal
of the underlying intervention.


The probability of obtaining this effect by chance is very small
(Bayesian one-sided tail-area probability p = 0.0).
This means the causal effect can be considered statistically
significant.
